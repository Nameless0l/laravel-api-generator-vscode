# Generate your first API

Open the generator form and describe your entity:

1. **Entity name** in PascalCase, e.g. `Product`, `BlogPost`
2. **Fields** with their types (`name:string`, `price:float`, …)
3. **Relationships** to other entities (hasMany, belongsTo, …)
4. **Options**: Sanctum auth, Postman collection, soft deletes,
   Spatie QueryBuilder filters

Click **Generate API** and you get the full stack in one shot:

Model · Migration · Controller · Service · DTO · Request ·
Resource · Policy · Factory · Seeder · Tests · Routes

Use **Preview Files** first if you want to inspect the code before
writing anything to disk.
